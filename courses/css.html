CSS Basics for Beginners

Introduction to CSS

CSS (Cascading Style Sheets) is used to style and format the appearance of HTML documents. It allows you to control the colors, layout, fonts, and more.

How CSS Works

CSS uses rules to target HTML elements and apply styles to them. A rule consists of a selector and a set of declarations.

Selector Types

  • Element Selector: Targets all instances of a specific HTML element (e.g., p { }).
  • Class Selector: Targets elements with a specific class attribute (e.g., .header { }).
  • ID Selector: Targets a single element with a specific ID attribute (e.g., #main-title { }).

Basic Syntax

            
    /* Selectors target HTML elements */
    selector {
        property: value;
    }
    
    /* Example */
    p {
        color: blue;
    }

Box Model

The Box Model consists of content, padding, border, and margin. Use properties like 'width', 'height', 'padding', 'border', and 'margin' to control an element's layout.

Units of Measurement

  • px (Pixels): Fixed size.
  • % (Percentage): Relative to the parent element.
  • em: Relative to the font-size of the element.

Colors

Colors can be specified using color names, hexadecimal values, RGB, and HSL.

Fonts

Use the 'font-family', 'font-size', 'font-weight', and 'text-align' properties to control text appearance.

Backgrounds

Change background colors and images with 'background-color' and 'background-image' properties.

Positioning and Layout

Use 'position' and 'display' properties to control how elements are positioned on the page.

Flexbox

Flexbox is a layout model for arranging elements in a flexible way within a container.

Grid Layout

Grid Layout is a two-dimensional layout system for arranging content into rows and columns.

Responsive Design

Use media queries to apply different styles based on screen sizes, making your site responsive.

CSS Selectors Specificity

If conflicting styles are applied, CSS uses specificity to determine which one to apply.

Transitions and Animations

Apply smooth transitions and animations using 'transition' and 'animation' properties.
